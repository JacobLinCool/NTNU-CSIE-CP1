# HW 2.6 - Bonus: Colorful Output

我們可以用 ANSI 跳脫序列 (ANSI Escape Sequences) 來改變輸出的表現風格，像是粗體及顏色等。

其結構為 `ESC[` 加上後方的一或多個以 `;` 分割的參數，並以 `m` 結尾。

在這個作業中，我們將使用 ANSI 跳脫序列讓輸出變成**粗體紅色文字**。

`\033` 代表的是 `ESC`，因為 `ESC` 不是可視字元。

```c
int main() {
    // 將風格設為租體(1)、紅色文字(31)
    printf("\033[1;31m");

    // 輸出要輸出的內容
    printf("Hello World\n");

    // 重設回初始的風格
    printf("\033[0m");

    return 0;
}
```
